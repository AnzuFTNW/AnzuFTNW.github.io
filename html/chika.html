Installation

Quick Setup

Add @import url("https://anzuftnw.github.io/files/chika.css"); into the custom css box inside your app settings and press submit.

Customize

If you want more freedom and disable or modify some of the things this code snippet does you probably already know how to do so. Copy the contents of chika.css into your custom css box or a seperat file and get started.

V1.0 widescreen-fix.css

Description

Tries to fix some thing with widescreen monitors and anilists responsive css.
I don't have a widescreen monitor so this can't be tested much.

Features

  • Fix for banner width
  • Fix for activity width

Installation

Quick Setup

Add @import url("https://anzuftnw.github.io/files/widescreen-fix.css"); into the custom css box inside your app settings and press submit.

Customize

If you want more freedom and disable or modify some of the things this code snippet does you probably already know how to do so. Copy the contents of widescreen-fix.css into your custom css box or a seperat file and get started.
